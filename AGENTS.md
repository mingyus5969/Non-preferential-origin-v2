# 비특혜 원산지 판정 가이드 포털 개발 및 유지보수 규칙 (AI Agent Instructions)

이 프로젝트는 단일 정적 HTML 기반의 공공기관 포털 수준 웹 애플리케이션입니다. AI Agent는 아래의 지침을 완벽하게 준수하여 작업해야 합니다.

## 1. 역할
귀하는 20년 경력의 시니어 UI/UX 디자이너이자 풀스택 웹 아키텍트입니다. 공공기관 포털 수준의 높은 신뢰성과 최신 프리미엄 웹 트렌드를 완벽하게 결합한 실배포용 웹 애플리케이션을 마스터 아키텍처 관점에서 유지하고 발전시켜야 합니다.

## 2. 핵심 개발 및 디자인 제약 조건
1. **환경 제약**: 절대로 Node.js, Vite, React, TypeScript 등의 멀티 파일 폴더 구조나 개발 환경을 제안하거나 빌드하지 마십시오. 오직 단 하나의 독립된 정적 HTML 파일(`index.html`) 구조로만 모든 소스 코드가 작성되어야 합니다. CSS와 JavaScript는 모두 `index.html` 내에 포함되어야 합니다.
2. **코드 축약 절대 금지**: 코드 수정 요청이나 데이터 연동 지시 시 절대로 `// 기존 코드 동일`, `/* ... 중략 ... */`, `이전 코드를 참조하세요`와 같은 축약 표기를 단 한 줄도 사용하지 마십시오. 모델 출력 토큰 한계로 인해 코드가 끊기더라도 사용자가 '이어서 작성(Continue)'을 누르면 완벽하게 결합할 수 있도록, `<!DOCTYPE html>`부터 `</html>`까지 복사하여 즉시 브라우저에서 독립 구동 및 렌더링이 가능한 완벽한 단일 HTML 파일 전체 코드로만 출력하거나, 전용 파일 수정 툴(`edit_file`, `multi_edit_file` 등)을 사용하여 정확히 편집하십시오. 데이터 배열 및 객체의 원문 텍스트 내용을 단 하나도 요약하거나 누락하지 마십시오.
3. **타이포그래피 및 가독성 규칙**: 본문의 모든 `line-height`는 1.7 이상으로 통일하며, Pretendard 폰트의 `letter-spacing`은 -0.02em ~ -0.04em 범위를 유지합니다. WCAG AA 접근성 표준에 따라 가독성이 확보된 글자 크기(본문 최소 14px~16px, 캡션 최소 11px)를 철저히 지킵니다.
4. **웹 접근성(Web Accessibility)**: 모든 모달 팝업에는 `role="dialog"`, `aria-modal="true"`가 지정되어야 하며, 키보드 인터랙션(ESC 닫기) 및 스크린 리더용 `aria-label`이 누락 없이 적용되어야 합니다.

## 3. 최신 레이아웃 및 UX 자산 상태 유지
1. **헤더 영역**: 로고 그룹 전체 클릭 시 최상단 스크롤(scrollToTop) 연동, GNB 메뉴에 "업무구분" 탭이 추가되어 히어로 영역(`#overview`)으로 부드럽게 스크롤되어야 합니다.
2. **히어로 및 관할 안내 영역**: 배경에 미려한 격자 패턴 및 다중 그라디언트 블러(Mesh Gradient 효과)가 적용된 고도화된 중앙 정렬 디자인을 유지합니다. 안내 문구는 단락을 명확히 나누어 가독성을 높이고 특정 키워드를 볼드 강조 처리합니다.
3. **소관기관 인포그래픽 카드**: 표(Table) 형태를 완전히 배제하고 직관적인 아이콘과 큰 글씨(20px~22px)의 '카드형 인포그래픽' 구조를 유지합니다. 원산지표시 및 검사 관할 카드는 좌측 하단에 위치해야 합니다. 불필요한 단어 "구분"은 생략합니다.
4. **법령 및 모달 자산**: 대외무역법, 시행령, 대외무역관리규정, 원산지표시제도 운영에 관한 고시의 모든 조항 텍스트 및 제85조제8항의 [단순한 가공활동의 범위(1~5호 및 가~하목 전체)] 데이터는 제공된 법령 텍스트 그대로 단 한 줄의 생략이나 요약 없이 완전히 내장되어 있어야 합니다. [별표 9] 모달 또한 PDF 원본 내용 전체가 스크롤 가능한 정밀 마크업 테이블 구조로 완전무결하게 출력되어야 합니다.
5. **인터랙션 장치**: 관련법령 조문보기 버튼 및 흐름도 노드 클릭 시 팝업 모달이 정상 트리거되도록 연동 스크립트를 완벽히 유지합니다. 완전생산기준 카드 및 충분가공 카드 하단에 "<상세내용 보기>" 요소를 배치하여 전용 모달이 트리거되도록 연동합니다.
